# 时钟
一个可以查看时间范围的时钟

## .timeRange不固定宽度变化对画布的影响
### 技术原理：DOM布局与重排
当DOM元素的尺寸或位置发生变化时，浏览器需要重新计算元素的布局，这个过程称为 重排（reflow） 。重排会影响：

- 元素的 getBoundingClientRect() 返回值
- 元素的位置和尺寸
- 父容器和兄弟元素的布局
在这个时钟应用中，频繁的重排导致canvas的边界信息不稳定，从而影响了鼠标坐标到角度的计算，诱发时钟抖动，影响用户体验。因此需要给父容器或.timeRange或canvas添加一个固定的宽度，避免触发重排。

## 关键技术点
1. 离屏渲染 ：使用离屏画布缓存静态元素
2. 分层设计 ：将绘制内容分为静态和动态两部分
3. 局部更新 ：只更新需要变化的部分
4. 性能优化 ：提升了整体性能，为用户提供了更加流畅的交互体验
    - 减少不必要的绘制操作
    - 静态元素只在必要时更新
    - 降低CPU和GPU使用率
